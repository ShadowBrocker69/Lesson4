git --version
Проверка версии Git и установлен ли он
git init
Инициализацяи нового репозитория. Один раз на проект в самом начале
git add index.html
добавление к коммиту файла index.html
git commit -m "пояснения к коммиту"
Создание нового коммита (новая запись в истории)
git status"
Показывает статус репозитория и есть ли что коммитить
gitk"
Открывает программу показывающую список коммитов
git remote add origin
Добавляет связь между локальным и удалённым репозиторием
git push
Отправляет изменения в удалённый репозиторий
